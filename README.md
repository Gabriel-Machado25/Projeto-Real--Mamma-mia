# 🍕 Mamma Mia Pizzaria - Sistema de Pré-Reserva

Este projeto é uma Landing Page desenvolvida para a **Mamma Mia Pizzaria**, focada em otimizar o processo de reservas para o rodízio da casa. A solução resolve o problema de comunicação lenta entre cliente e atendente, automatizando a formatação dos dados.

## 🚀 O Problema vs. A Solução
Antes, o cliente iniciava uma conversa genérica no WhatsApp, gerando várias trocas de mensagens para coletar dados básicos. 
**Agora:** O cliente preenche um formulário intuitivo e o sistema gera uma mensagem pré-formatada profissional, enviando-a diretamente para o atendente com todos os detalhes (nome, data, pessoas, observações).

## ✨ Funcionalidades
- **Formulário Inteligente:** Captura dados essenciais para o rodízio.
- **Automação de Mensagem:** Integração com a API do WhatsApp para envio de dados estruturados.
- **Design Responsivo:** Totalmente adaptado para dispositivos móveis (foco do público-alvo).
- **UX Otimizada:** Mensagem clara que facilita a leitura imediata pelo atendente.

## 🛠️ Tecnologias
- **HTML5** & **CSS3** (Layout e estilização moderna)
- **JavaScript (ES6+)** (Manipulação de DOM e lógica de integração com WhatsApp)
- **Vercel** (Hospedagem e Deploy contínuo)

## 📋 Como funciona
1. O usuário acessa a Landing Page.
2. Preenche as informações de sua preferência para o rodízio.
3. Ao clicar em "Reservar", o script JS formata os dados e abre o WhatsApp com a mensagem pronta:
> *NOVA RESERVA — Mamma Mia Pizzaria*
> 👤 *Cliente:* [Nome]
> 👥 *Pessoas:* [Quantidade]
> ...
